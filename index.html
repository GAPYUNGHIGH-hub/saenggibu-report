A 평가관 — 학업역량·탐구역량
교과 세특 / 수행평가 / 교과 간 융합 중점 심사
"교과 심화의 폭은 넓지만, 깊이의 증거가 불균형하다. 수학·과학은 강하고, 국어·사회는 탐구보다 활동 서술에 머문다."
강점 — 확실히 드러난 것
  • 수학 세특: 혈당변화 이차함수 모델링, 라돈 42일 데이터 일차함수 시각화 — 실데이터와 교과 개념을 직접 연결한 구체적 탐구역량. 수학 세특 중 최상위 수준.
  • 과학탐구실험 세특: 접지·우유부패 실험 설계 → 변수 통제 한계 자기 지적 → 논문 분석(노이즈 방전 170배) → 전문용어 습득. '예상 밖 결과 → 탐구 방향 전환'이 명확히 보이는 유일한 세특.
  • 정보 세특: 교과 탐구 결과를 직접 프로그램으로 구현(라돈 환기 알림, 모듈러 연산). 배운 것을 도구로 만든 사례 — 정보처리역량의 실증.
  • 교과 간 융합: 수학(함수) + 과학(라돈) + 정보(코딩) + 진로활동이 하나의 탐구 스토리로 연결. 드물고 강력한 서사.
약점 — 냉정한 지적
  • 국어 세특: 시 창작·독서일지 수준에 머묾. 진달래꽃 다학문 분석은 흥미롭지만 '수학으로 감정 그래프화'는 교과 연계가 아닌 아이디어 연상 수준.
  • 통합사회 세특: 발표·조사 활동 나열. '의료공백 시장실패 분석'은 방향이 좋으나 본인의 탐구 과정·난관·깨달음이 없다. 선생님이 요약한 내용이지 학생이 고민한 흔적이 보이지 않음.
  • 한국사 세특: 다양한 활동(가네코 후미코, 가상판결문, 영화기획)이 진로와 연결되지 않음. 역사 교과에서 간호·의료와 연결하는 탐구 시도 전무.
  • 전체 난관·극복 서사 부재: 과학탐구실험을 제외한 모든 세특에서 '막혔다 → 방법을 바꿨다 → 새로운 깨달음'의 구조가 없다. 탐구가 아닌 수행 결과 보고에 가깝다.

B 평가관 — 진로역량·자기주도성
진로 일관성 / 자기주도 탐구 / 진로 연결 필연성 중점 심사
"진로 키워드(간호·뇌신경)는 전방위 등장하지만, '왜 이 학생이어야 하는가'의 고유한 답이 아직 없다. 두 탐구 스토리가 분산돼 있어 통합이 시급하다."
강점 — 진로 서사의 씨앗
  • 접지×산화환원×간호 축: 통합과학·과탐실험·스펙트럼동아리·자율활동·정보 세특이 연결. '접지 → 항산화 → 의료기기 노이즈 → 환자 안전'까지 이어지는 심화 구조 형성 시작.
  • 명상×뇌과학×전인간호 축: 인문독서발표·통합과학·영어 세특에서 연기론·신경가소성·감정재해석까지 연결. 철학-과학-임상을 잇는 독특한 진로 스토리 가능성.
  • 데이터과학 도구 능력: 라돈 42일 측정·파이썬 프로그램·PEF 생체지표 측정 — 간호학에서 보기 드문 이공계 데이터 리터러시. 뇌신경 전문간호사 진로와 연결 시 강력한 차별점.
  • AI 의료 토론: 단순 찬반이 아닌 '단계적 도입 전략'이라는 구체적 대안 제시. 비판적 사고 역량이 진로 맥락에서 드러남.
약점 — 진로 서사의 치명적 문제
  • 두 축의 연결점이 없다: '접지·산화환원' 축과 '명상·뇌과학' 축이 1학년 생기부 어디에도 만나지 않는다. 현재는 두 명의 다른 학생처럼 읽힌다. 2학년에서 반드시 통합해야 한다.
  • 진로 연결이 결론부 선언에 집중: "간호사가 되겠다", "간호사에게 ~이 중요하다" 식의 선언이 대부분. 탐구 과정 중간에서 진로가 자연스럽게 불거지는 장면이 없다.
  • 코티솔 독서 세특 아쉬움: 코티솔과 뇌신경·명상을 연결할 절호의 기회인데 '심층 탐구' 수준에서 끝남. 논문 분석·자기 측정 실험 없이 독서 요약+적용 선언으로 마무리.
  • 자기주도 탐구 증거 부분적: 과탐실험·정보·진로활동은 자기주도적이나, 나머지 교과 세특은 수업 참여·발표 수준. 스스로 연구 문제를 설정하고 방법을 결정한 장면이 부족하다.

C 평가관 — 공동체역량·인성·리더십
창체활동 / 봉사 / 리더십 / 공동체 기여 중점 심사
"착하고 성실한 학생이라는 인상은 매우 강하다. 그러나 성실함이 리더십의 증거가 되려면 '문제를 발견하고 구조를 바꾼' 장면이 필요하다."
강점 — 확실히 좋은 것
  • 부회장 리더십: 스포츠 축제 티셔츠 구입 주도, 치어리딩 안무 완성 — 구체적 행동 서술. 형식적 직함이 아닌 실제 역할이 기록됨.
  • 봉사 다양성: 배식·청소·공동체학습 도우미·합창발표회 기획·환경교육봉사·수능고사장 정돈 — 총 21시간, 의무적 봉사가 아닌 자발적 참여 다수.
  • 교과도우미 역할 다수: 한문·체육1 교과 도우미, 공동체학습 도우미 — 공동체 기여 의식이 일관되게 보임.
  • 1학기 모범학생 표창: 공식 인정. 인성 영역의 강력한 외적 증거.
약점 — 깊이의 문제
  • 봉사가 '돕는 역할'에 집중: 모든 봉사가 기존 구조 안에서 보조하는 형태. 본인이 문제를 발견하고 프로그램을 기획한 봉사가 한 건도 없다.
  • 리더십의 구체성 부족: 부회장으로서 갈등 조정, 소외 학생 포용, 제안 채택 같은 '관계 리더십'의 장면이 없다. '업무 집행형' 리더십만 기록됨.
  • 공동체 탐구 연결 미흡: 공기질 개선 프로젝트가 사회 기여 탐구로 발전할 가능성이 있었으나, 실제 학급 환경 개선 제안·실행 기록이 없다.

최우선 / 구조적 결함
난관·극복·깨달음 서사 전면 부재
과탐실험 세특 외 전 교과에서 '막혔다 → 스스로 해결했다 → 새롭게 알았다'의 구조가 없다. 입학사정관은 이 서사에서 학생의 탐구 진정성을 판단한다. 현재 생기부는 대부분 '수행 결과 보고'이지 탐구 과정 기록이 아니다. 2학년 전 세특에서 이 구조를 의도적으로 설계해야 한다.
2순위 / 진로 서사 분열
'접지·산화환원' 축과 '명상·뇌과학' 축이 어디서도 만나지 않음
1학년 생기부는 두 개의 독립된 탐구 계열이 공존하는 상태다. 뇌신경 전문간호사라는 진로 아래 두 축이 수렴해야 비로소 '설계된 학생'으로 읽힌다. 2학년에서 두 축이 만나는 통합 탐구 1건이 반드시 완성되어야 한다. (예: 접지가 뇌파에 미치는 영향, 명상 전후 산화 스트레스 측정)
3순위 / 인문사회 공백
국어·한국사·사회 세특이 탐구가 아닌 참여·발표 수준
이과형 교과의 탐구 깊이와 문과형 교과의 탐구 깊이가 극명하게 차이 난다. 입학사정관은 전 교과에서 고르게 탐구하는 학생을 선호한다. 국어(의료 문학·생명윤리), 한국사(한국 의료사), 사회(의료 불평등 정책 설계) 각 1건의 심화 탐구가 2학년에서 필요하다.

2학년 집중 강화 전략 — 5가지 실행 과제
1
최우선 — 전 교과 적용
모든 탐구에 난관·극복·깨달음 구조를 의도적으로 설계
수행평가 시작 전 "어디서 막힐 것인가"를 먼저 예측하고, 그 지점에서 방법을 바꾸는 행동을 기록으로 남긴다. 깨달음은 반드시 "이 탐구 전에는 몰랐던 것"이어야 한다. 선생님이 세특을 쓸 때 이 3가지 장면이 자연스럽게 들어가도록 결과물을 역설계해야 한다.
2
두 탐구 축을 '생체 데이터 × 뇌신경 × AI 간호'로 통합하는 탐구 1건 완성
접지(전위안정)가 뇌파(뇌신경)에 미치는 영향, 명상 전후 산화환원 지표 변화 등 두 축이 만나는 지점을 탐구한다. 이것이 완성되면 1~2학년 전체 생기부가 하나의 서사로 읽힌다.
3
국어·한국사·사회 세특에서 각 1건의 진로 연결 심화 탐구 확보
국어: 의료 문학(간호 서사) 분석 + 생명윤리 텍스트 비판적 읽기. 한국사: 한국 의료史와 현재 의료격차 연결. 사회: 의료 불평등을 시장실패 개념으로 분석하고 본인의 정책 대안 제시. 인문교과도 '탐구하는 학생'으로 읽혀야 한다.
4
자기 몸을 대상으로 한 자기추적 실험 1건 공식화
명상 전후 심박수·PEF·집중력 점수를 직접 측정하고 데이터를 축적한다. 이미 PEF·SpO2 장비가 있으므로 측정 비용 없음. 세특에 "직접 실험 설계·측정·분석"이 기록되면 탐구역량이 최상위 수준으로 격상된다.
5
공동체 기여 탐구를 '직접 기획한 캠페인'으로 전환
교실 공기질 데이터를 토대로 '9시간 환기 캠페인'을 학교 내에서 실제 실행하고 전후 데이터를 비교한다. 봉사가 '돕는 역할'에서 '변화를 만드는 역할'로 격상되면 공동체역량과 탐구역량이 동시에 강화된다.
